кнопка >> развернуть

далее кнопки - часть интуитивная

net - переключение в режим сетки и заливки (как будет в игре), текстура пока левая

add - добавляет новую фигуру

мышкой drag&drop можно таскать экран влево-вправо

колесо - масштаб

если зажать Shift - можно таскать отдельные точки фигуры мышкой drag&drop
(точность наведения на точку очень высокая - проще на большом масштабе)

Shift + Alt + мышкой drag&drop за любую точку - перещаем всю фигуру целиком

Ctrl + клик по линии - создать новую точку

D + клик по точке - удалить точку
