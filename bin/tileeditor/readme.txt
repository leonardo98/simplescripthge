кнопка >> развернуть

далее кнопки - часть интуитивная

net - переключение в режим сетки и заливки (как будет в игре), текстура пока левая

add - добавляет новую фигуру, мышкой можно таскать экран влево-вправо

колесо - масштаб

если зажать Shift - можно таскать точки мыкой(точность наведения на точку очень высокая - проще на большом масштабе)

Shift + Alt + мышкой за любую точку - перещаем фигуру

Ctrl + клик по линии - создать новую точку

D + клик по точке - удалить точку

Shift+D+click по точке на линии - удалить всю линию

Delete - удалить активное украшение/накладку
